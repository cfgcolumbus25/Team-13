Some features implemented:
- Input scores for up to 33 CLEP exams and see how they translate into credits at different colleges.
- Filter colleges by ZIP code and set a search radius to find nearby institutions.
- Colleges are ranked by total credits awarded, with distance used as a tiebreaker for better recommendations.
- Matching CLEP exams are highlighted in green for quick reference.
- Each college card shows course equivalencies, total credits, and distance from your location.
- Fully designed interface with collapsible filters, modern layout, and mobile-friendly components.
- Uses the Haversine formula for precise distance measurements between ZIP codes.
