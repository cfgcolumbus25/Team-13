# Backend Setup & Local Development

This backend is an Express.js server that connects to Supabase. Follow the steps below to run it locally.

---

## Prerequisites

- [Node.js](https://nodejs.org/) (LTS recommended)
- [npm](https://www.npmjs.com/) (comes with Node)
- A Supabase project (for database + API keys)

---

## 1. Navigate to the Backend Folder

From the project root:

```bash
cd backend
```

## 2. Install Dependencies
```bash
npm install
```

## 3.  Create .env file
Replace the following environmental variables with the keys from the google doc
```bash
SUPABASE_ANON_KEY=
SUPABASE_URL=
```


## 4.  Start server
Replace the following environmental variables with the keys from the google doc
```bash
node server.js
```
